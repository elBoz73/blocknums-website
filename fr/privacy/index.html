Politique de confidentialité

BlockNums est conçu pour faire le moins possible avec vos données. Cette page résume ce qui reste sur votre appareil, ce qui est envoyé sous forme anonyme, et ce qui n'est jamais collecté.

Ce qui reste sur votre appareil

Listes de blocage, liste VIP et historique d'appels (selon la durée que vous choisissez) sont stockés localement, dans la base de données chiffrée de l'application.

BlockNums n'héberge aucun serveur qui collecterait ces contenus.

Permissions Android

Filtrage d'appels — nécessaire pour que BlockNums puisse filtrer les appels entrants (rôle système, pas une permission classique).

Contacts (facultatif) — pour afficher un nom lors du filtrage et choisir des numéros dans vos listes.

Notifications (facultatif) — pour vous alerter après qu’un appel a été bloqué.

État du téléphone — utilisé uniquement sur l'appareil pour identifier les appels entrants à filtrer.

Ces permissions n'envoient pas vos listes, contacts ou historique vers un serveur BlockNums.

Statistiques anonymes (Firebase Analytics)

Les statistiques sont désactivées par défaut. Elles ne sont envoyées qu'après activation explicite dans Réglages → Confidentialité.

Lorsqu'elles sont activées, l'application peut envoyer quelques événements techniques anonymes : `blocked_added`, `pattern_added`, `whitelist_added`, `limit_reached`, `purchase_success`.

Les paramètres associés sont strictement non identifiants : un mode d'affichage, un identifiant produit Google Play, un compteur agrégé après une action, des indicateurs simples (par exemple la présence d'une catégorie), le nom d'une fonctionnalité lorsqu'un plafond est atteint.

Aucun de ces événements ne contient de numéro de téléphone (même partiel), la chaîne d'un modèle de blocage, un libellé ou une description saisis par vous, le contenu de vos contacts, le texte des SMS ni l'audio des appels.

Ce que nous ne collectons pas

Vos numéros de téléphone (saisis ou reçus).

Le contenu de votre répertoire ou de vos SMS.

Le son ou l'enregistrement de vos appels.

Aucun identifiant publicitaire pour cibler de la publicité.

Achats Premium

Les paiements sont traités par Google Play. Google peut collecter les informations liées au paiement selon ses propres règles. BlockNums ne reçoit que la confirmation que vous êtes bien titulaire d'un achat Premium.

Enfants & familles

BlockNums est un outil de gestion des appels indésirables. Il n'est pas conçu pour les enfants de moins de 13 ans.

Les adolescents (16–17 ans) peuvent l'utiliser sur leur téléphone ; une supervision parentale est recommandée pour les achats intégrés.

L'application ne contient aucune publicité et ne profile pas les utilisateurs à des fins publicitaires.

Vos droits & suppression des données

Vous pouvez désactiver les statistiques anonymes à tout moment depuis Réglages → Confidentialité, sans impact sur le fonctionnement.

Désinstaller l'application ou effacer ses données dans les paramètres Android supprime les listes, la VIP et l'historique locaux sur l'appareil.

Pour toute demande RGPD ou exercice de droit : fiche Play Store ou issue GitHub (elBoz73/blockNums).

Pour toute question relative à la confidentialité ou aux données : fiche BlockNums sur Google Play, ou GitHub (elBoz73/blockNums).
