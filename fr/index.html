Guide d'utilisation

Tout ce qu'il faut savoir, en 7 étapes courtes.

  1. 1/7

    1. Installer BlockNums

    Installez l'application depuis Google Play. Aucun compte n'est demandé : il n'y a rien à créer.

    Lancez l'application puis suivez l'onboarding rapide. Tout se passe sur votre appareil.

  2. 2/7

    2. Définir BlockNums comme appli de filtrage par défaut

    Android demande votre accord pour qu'une application décide automatiquement quels appels filtrer. Acceptez quand BlockNums vous le propose.

    Sans cet accord, l'application ne peut filtrer que manuellement (vous pouvez le réactiver à tout moment depuis Réglages → Filtrage par défaut).

  3. 3/7

    3. Créer votre liste de blocage

    Onglet Blocages : ajoutez les numéros que vous voulez voir refusés automatiquement. La version gratuite autorise jusqu'à 30 numéros.

    Vous pouvez aussi créer des catégories (par exemple « Démarchage », « Arnaques connues ») pour vous y retrouver.

  4. 4/7

    4. Modèles de blocage (préfixes, indicatifs)

    Un modèle bloque toute une plage de numéros, par exemple `0899*` ou `+33899*`. La version gratuite inclut 2 modèles simples.

    Le mode regex avancé (motifs complexes) est inclus dans Premium. Vos modèles existants ne sont jamais perdus.

  5. 5/7

    5. Liste VIP (Premium)

    La VIP garantit qu'un appel important passe toujours, y compris si vous activez un mode strict (par exemple « Tout bloquer »).

    Idéal pour vos proches, votre médecin ou vos contacts professionnels indispensables.

  6. 6/7

    6. Premium et restauration

    Premium est un achat unique sur Google Play. Pas d'abonnement.

    Si vous changez de téléphone ou réinstallez : Réglages → Premium → Restaurer mon achat. Connectez-vous au même compte Google.

  7. 7/7

    7. Dépannage

    Un appel n'a pas été bloqué ? Vérifiez que BlockNums est bien défini comme application de filtrage par défaut, et que la règle correspond bien (numéro complet, modèle, etc.).

    Trop d'appels bloqués ? Consultez l'historique pour identifier la règle concernée et l'ajuster (ou ajoutez le contact à la liste VIP).

    Les statistiques anonymes peuvent être désactivées depuis Réglages, sans impact sur le fonctionnement.

Questions fréquentes

Les vraies questions que vous nous posez.

Mes numéros de téléphone sont-ils envoyés à un serveur ?+

Non. Vos listes de blocage, votre liste VIP et votre historique d'appels sont stockés sur votre appareil. BlockNums n'héberge aucun serveur qui collecterait ces contenus.

Quelles statistiques anonymes sont envoyées ?+

Pour mesurer l'utilisation des fonctionnalités et améliorer le produit, l'application peut envoyer quelques événements techniques anonymes via Firebase Analytics : blocked_added, pattern_added, whitelist_added, limit_reached, purchase_success (voir la politique de confidentialité sur cette page).

Les paramètres sont strictement non identifiants : mode d'affichage, identifiant produit Google Play, compteur agrégé après une action, indicateurs simples (par exemple la présence d'une catégorie), nom de la fonctionnalité lorsqu'un plafond gratuit est atteint.

Aucun de ces événements ne contient de numéro de téléphone (même partiel), la chaîne d'un modèle de blocage, un libellé ou une description saisis par vous, le contenu de vos contacts, le texte des SMS ni l'audio des appels.

Vous pouvez désactiver l'envoi de ces statistiques dans Réglages → Confidentialité, comme décrit dans les mentions légales.

Premium est-il à vie ?+

Oui. Premium est un achat unique sur Google Play, pas un abonnement. Vous gardez les fonctionnalités premium tant que vous utilisez l'application avec le même compte Google.

Comment restaurer un achat Premium ?+

Réglages → Premium → Restaurer mon achat. Connectez-vous avec le même compte Google que celui utilisé pour l'achat initial.

Pourquoi un appel n'a-t-il pas été bloqué ?+

Plusieurs raisons possibles : BlockNums n'est pas l'application de filtrage par défaut, la règle ne correspond pas exactement au numéro reçu (préfixe, indicatif), ou l'opérateur a transmis un identifiant masqué.

Consultez l'historique d'appels dans l'app pour identifier la situation et ajustez votre règle.

Compatible avec mon téléphone ?+

BlockNums fonctionne sur Android. La fonctionnalité de filtrage automatique nécessite Android 9 ou supérieur (API 28) et que vous l'autorisiez comme app de filtrage par défaut.

L'application est-elle open source ?+

Non. BlockNums est un logiciel propriétaire : son code source n'est pas publié. Les bibliothèques tierces intégrées restent sous leurs propres licences (voir Réglages → Licences tierces).

Que se passe-t-il si je désinstalle BlockNums ?+

Toutes vos données locales sont supprimées avec l'application (listes, historique, paramètres). Réinstallez et restaurez Premium si besoin.

Politique de confidentialité▾

BlockNums est conçu pour faire le moins possible avec vos données. Cette page résume ce qui reste sur votre appareil, ce qui est envoyé sous forme anonyme, et ce qui n'est jamais collecté.

Ce qui reste sur votre appareil

Listes de blocage, liste VIP et historique d'appels (selon la durée que vous choisissez) sont stockés localement, dans la base de données chiffrée de l'application.

BlockNums n'héberge aucun serveur qui collecterait ces contenus.

Permissions Android

Filtrage d'appels — nécessaire pour que BlockNums puisse filtrer les appels entrants (rôle système, pas une permission classique).

Contacts (facultatif) — pour afficher un nom lors du filtrage et choisir des numéros dans vos listes.

Notifications (facultatif) — pour vous alerter après qu’un appel a été bloqué.

État du téléphone — utilisé uniquement sur l'appareil pour identifier les appels entrants à filtrer.

Ces permissions n'envoient pas vos listes, contacts ou historique vers un serveur BlockNums.

Statistiques anonymes (Firebase Analytics)

Les statistiques sont désactivées par défaut. Elles ne sont envoyées qu'après activation explicite dans Réglages → Confidentialité.

Lorsqu'elles sont activées, l'application peut envoyer quelques événements techniques anonymes : `blocked_added`, `pattern_added`, `whitelist_added`, `limit_reached`, `purchase_success`.

Les paramètres associés sont strictement non identifiants : un mode d'affichage, un identifiant produit Google Play, un compteur agrégé après une action, des indicateurs simples (par exemple la présence d'une catégorie), le nom d'une fonctionnalité lorsqu'un plafond est atteint.

Aucun de ces événements ne contient de numéro de téléphone (même partiel), la chaîne d'un modèle de blocage, un libellé ou une description saisis par vous, le contenu de vos contacts, le texte des SMS ni l'audio des appels.

Ce que nous ne collectons pas

Vos numéros de téléphone (saisis ou reçus).

Le contenu de votre répertoire ou de vos SMS.

Le son ou l'enregistrement de vos appels.

Aucun identifiant publicitaire pour cibler de la publicité.

Achats Premium

Les paiements sont traités par Google Play. Google peut collecter les informations liées au paiement selon ses propres règles. BlockNums ne reçoit que la confirmation que vous êtes bien titulaire d'un achat Premium.

Enfants & familles

BlockNums est un outil de gestion des appels indésirables. Il n'est pas conçu pour les enfants de moins de 13 ans.

Les adolescents (16–17 ans) peuvent l'utiliser sur leur téléphone ; une supervision parentale est recommandée pour les achats intégrés.

L'application ne contient aucune publicité et ne profile pas les utilisateurs à des fins publicitaires.

Vos droits & suppression des données

Vous pouvez désactiver les statistiques anonymes à tout moment depuis Réglages → Confidentialité, sans impact sur le fonctionnement.

Désinstaller l'application ou effacer ses données dans les paramètres Android supprime les listes, la VIP et l'historique locaux sur l'appareil.

Pour toute demande RGPD ou exercice de droit : fiche Play Store ou issue GitHub (elBoz73/blockNums).

Pour toute question relative à la confidentialité ou aux données : fiche BlockNums sur Google Play, ou GitHub (elBoz73/blockNums).
