Deploiement GitHub Pages — fichier OBLIGATOIRE

Copiez TOUT le contenu de out/ a la racine du repo blocknums-website.

IMPORTANT : le fichier .nojekyll (a la racine) doit etre present.
Sans lui, GitHub ignore le dossier _next/ et le CSS ne charge pas.

Sous Windows, activez "Elements masques" dans l explorateur pour le voir.
Ou creez-le sur GitHub : Add file > Create new file > nom : .nojekyll (vide)
